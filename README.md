# Swarm Formation Studio

A recruiter-friendly multi-agent simulation that demonstrates boid-style intelligence, control systems, and interactive mission shaping.

## Highlights
- Alignment, cohesion, and separation controls.
- Dynamic formation modes: free swarm, circle, line, and V formation.
- Real-time obstacle pressure response driven by mouse position.
- Live telemetry for speed, formation quality, and pressure.

## Stack
- HTML5 Canvas
- Vanilla JavaScript simulation loop
- Responsive CSS control surface

## Run
Open `index.html` in your browser.
